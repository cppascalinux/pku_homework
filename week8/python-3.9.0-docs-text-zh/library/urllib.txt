"urllib" --- URL 处理模块
*************************

**源代码:** Lib/urllib/

======================================================================

"urllib" 是一个收集了多个涉及 URL 的模块的包：

* "urllib.request" 打开和读取 URL

* "urllib.error" 包含 "urllib.request" 抛出的异常

* "urllib.parse"  用于解析 URL

* "urllib.robotparser" 用于解析 "robots.txt" 文件
